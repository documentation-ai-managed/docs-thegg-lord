## Recent Updates

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "bugfix"]}>

## New Features

- Added real-time collaboration, allowing multiple users to edit documents simultaneously with live cursors and conflict resolution.
- Introduced advanced search with full-text indexing and filters for tags, authors, and dates.
- New export options including PDF with custom branding and Markdown bundles.

## Improvements

- Enhanced performance of the document editor by optimizing rendering for large pages (`<50ms` load times).
- Updated UI with brand color `#3B82F6` accents for better visual consistency.

## Bug Fixes

- Fixed synchronization issues in shared workspaces that caused outdated content displays.
- Resolved export errors for documents containing embedded images or `{code blocks}`.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "improvement"]}>

## New Features

- Dark mode support with automatic detection based on system preferences.
- Integrated version history with diff views and one-click restores.

## Improvements

- Streamlined onboarding flow, reducing setup time by 40%.
- Added keyboard shortcuts for common actions like `<kbd>Ctrl</kbd>+<kbd>S</kbd>` to save.

## Bug Fixes

- Corrected rendering glitches in Mermaid diagrams on mobile devices.
- Patched authentication token refresh to prevent logout loops.

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## Initial Release

- Core documentation editor with rich text, MDX support, and live preview.
- Workspace organization with folders, tags, and permissions.
- Basic API for programmatic document management.

## Breaking Changes

- Initial schema update requires re-importing legacy documents.

</Update>

## Upgrade Process

Stay current by following these steps in your dashboard.

<Steps>
  <Step title="Check Current Version" icon="info">
    Navigate to **Settings {arrow-right} About** to view your installed version.
  </Step>
  <Step title="Review Changes" icon="book-open">
    Scan this changelog for breaking changes relevant to your workflows.
  </Step>
  <Step title="Apply Update" icon="download">
    Click **Update Now** in the dashboard. Backups are automatic.
  </Step>
  <Step title="Verify" icon="check-circle">
    Test key features like collaboration and search after updating.
  </Step>
</Steps>

<Callout kind="tip">
  Enable auto-updates in **Settings {arrow-right} Notifications** to receive new versions seamlessly.
</Callout>
